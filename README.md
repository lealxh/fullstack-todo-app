# Todos APP

Todos manager with backend and frontend in next.js, and redux to manage state

## Usage/Examples

- npm install
- npm run dev

Go to http://localhost:3000/

### Tech Stack

- Next.js
- React
- Redux
- Redux Toolkit
- Html
- CSS
